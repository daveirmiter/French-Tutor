📲 Install as App

Installing gives you a home screen icon, fullscreen mode, and persistent storage — your API key won't disappear between sessions.

  1. Host this file at a free HTTPS URL — easiest is tiiny.host: drag the file in, get a link in 30 seconds.
  2. Open that link in Chrome on your Android phone.
  3. Chrome will show an "Add to Home screen" banner — tap it. Or tap ⋮ menu → Add to Home screen.
  4. Tap Install. Marie appears as an app with persistent storage.
For a permanent free link: Create a GitHub account → New repository → upload the file → Settings → Pages → enable.
